CSDT\CollectionsBundle\CollectionHelper\HelperInterface\CollectionHelperInterface::mergeCollections

Merge collection

Return a collection of the same type as $collection1 with all the content of the second collection. Note a data lost is possible by example if you merge a ValueCollection as $collection1 and a MapCollection as $collection2.
